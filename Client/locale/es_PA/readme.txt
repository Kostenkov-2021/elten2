Bienvenido a Elten!
Este documento contiene algunos consejos e informaciones relacionadas con esta aplicación.

1. Requisitos del sistema
Sistema operativo: Windows 7 o superior
RAM: 512 MB
Procesador: 800MHz
Disco Duro: 256 MB de espacio libre
Acceso a internet

2. información preliminar
El autor del programa es  Dawid Pieper. Es una aplicación de código abierto, cuyo código se puede encotrar en el repositorio de gitHub en:
http://github.com/dawidpieper/elten2
Los detalles de licencia de este software están contenidos en un documento aparte.
Elten es una red social desarrollada especialmente para personas ciegas. consiste en un servidor, página de internet, y sobre todo, un cliente diseñado para Windows.
adicionalmente, es una api para  desarrolladores que escriben programas para ciegos.

3. Registro e inicio de sesión
 se puede acceder tanto al menú de registro como al de inicio de sesión desde la ventana de bienvenida. Recuerda escoger una contraseña atípica de forma que los intrusos no puedan acceder a tu cuenta.
Es importante escribir una dirección de correo real, porque ésta se puede usar para recuperar una contraseña olvidada.
Después de entrar exitosamente a tu cuenta, puedes habilitar el inicio de sesión automático,de forma que no necesites ingresar tu usuario y contraseña cada vez que ejecutes Elten.

4. Funciones básicas
En primera instancia, Elten es una red social. Se puede acceder a sus funciones principales desde el menú disponible presionando la tecla alt en la ventana principal, después de iniciar sesión.
El menú comunidad, como su nombre lo dice, contiene funciones para comunicarse con otros usuarios de Elten.
Aquí puedes encontrar funciones como foro, blogs o mensajes privados.
Deberías considerar llenar tu perfil en el submenú mi cuenta.
También puedes escribir algo sobre ti en tu tarjeta de visita.
El segundo menú se llama medios. Aquí puedes encontrar material compartido por otras personas, así como también navegar por YouTube.
Archivos es un simple administrador de archivos. Aquí puedes editar archivos de texto, reproducir música y, si quieres, subir tu avatar. Un avatar es una grabación de sonido disponible para ser reproducida por otros usuarios en el menú de tu perfil. también puedes subir algunos archivos al servidor, de forma que estén disponibles para ser descargados por otros usuarios de Elten.
El menú programas contiene aplicaciones externas escritas en la API de Elten.
Herramientas es un submenú que, como su nombre lo dice, contiene herramientas desarrolladas con el fin de administrar Elten. aquí puedes probar tu conexión a internet o acceder a la consola, así como también generar un nuevo tema de sonido o actualizar Elten a la versión más reciente.
configuraciones contiene diferentes opciones de configuración. Puedes considerar por ejemplo, esconder la ventana de Elten automáticamente en la bandeja de sistema. Esta opción se puede encontrar en las opciones de interfaz.
El menú ayuda incluye documentos relacionados con el uso del programa. aquí puedes encontrar la lista de cambios, así como la lista de atajos de teclado, licencia y este documento. También puedes reportar un error a través de este menú.
El último menú, salir, te permite ocultar o cerrar la aplicación.
